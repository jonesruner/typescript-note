## 概述
1. TypeScript 增加了什么？ 🔰
   1. 类型检测
   2. 支持Es新特性
   3. 丰富的配置项
   4. 添加ES不具备的新特性
2. TypeScript 环境
   1. 全局安装 **TypeScript**
        ```shell
        npm i typescript -g
        ```
   2. 创建一个ts文件
   3. 使用tsc对ts文件进行编译
        ```shell
        tsc *.ts  # 输入命令后 会生成一个 *.js 文件
        ```
